### Task Management
- Create, edit, and delete chores

- Set chore priority  
- Filter chores by status, user, or priority  
- Remove completed or canceled chores  
- Export and import chores to/from CSV files

### User Management
- Add multiple users  
- Assign tasks to individuals

### Completion
